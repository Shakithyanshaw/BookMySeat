# 🎬 QuickShow – Full Stack Movie Ticket Booking App

A full-stack Movie Ticket Booking Web Application built using the MERN stack.  
Users can browse movies, select seats, make payments, and receive booking confirmations.  
Includes an Admin Dashboard for managing movies, shows, and bookings.

## ✨ Features

### 👤 User Features

- User Signup/Login (Email, Social, Phone – via Clerk)
- Browse Movies
- View Movie Details & Showtimes
- Interactive Seat Selection
- Secure Online Payment (Stripe)
- Booking Confirmation Email
- Reminder Email Before Showtime
- View Booking History
- Add to Favorites

### 👨‍💼 Admin Features

- Admin Dashboard
- Add / Edit / Delete Movies
- Manage Shows & Showtimes
- View & Manage Bookings

---

## 🔥 Advanced Functionalities

### 🪑 Seat Reservation Logic

- Seats are reserved for 10 minutes if payment fails.
- If payment is not completed within 10 minutes, seats are automatically released.

### 💳 Stripe Payment Integration

- Secure payment processing
- Webhook integration for payment confirmation

### 📧 Email Automation (Inngest)

- Send booking confirmation email
- Send reminder email before showtime
- Notify users when a new movie is added
